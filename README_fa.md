## این پروژه چیه؟

توی ژوئن ۲۰۲۶، تلگرام یه ویرایشگر متن غنی (Rich Text Editor) معرفی کرد، اما فقط برای کاربران Premium. همزمان Bot API 10.1 رو منتشر کرد که دقیقاً همین قابلیت رو برای بات‌ها کاملاً رایگان در اختیار گذاشت.

این پروژه یه ویرایشگر بصری (WYSIWYG) می‌سازه که هر کسی -- نه فقط مشترکین Premium -- بتونه پیام‌های حرفه‌ای و قالب‌بندی شده بسازه و از طریق Bot API تلگرام ارسال کنه.

بدون نیاز به کدنویسی. بدون نیاز به Premium. بدون نیاز به سرور.

## قابلیت‌ها

### قالب‌بندی درون‌خطی
- پررنگ، کج، زیرخط، خط‌خورده
- اسپویلر، هایلایت، کد درون خطی
- زیرنویس، بالانویس
- لینک، اشاره به کاربر (منشن)
- پانویس با شماره‌گذاری خودکار

### المان‌های بلوکی
- سرتیتر (H1 تا H6)
- نقل‌قول با ذکر منبع
- نقل‌قول برجسته (Pull Quote)
- بلوک کد با تعیین زبان
- خط جداکننده
- بلوک تاشو (جزئیات)
- جدول با سرستون

### لیست‌ها
- لیست گلوله‌ای
- لیست شماره‌دار
- چک‌لیست (انجام‌شده/انجام‌نشده)

### رسانه
- تصویر (کشیدن و رها کردن یا آدرس)
- ویدیو، صدا، ویس نوت
- اسلایدشو با دکمه‌های مرور
- نقشه OpenStreetMap

### ریاضیات
- فرمول درون خطی (دلار دلار)
- فرمول بلوکی (دلار دلار)

### اتصال به API
- ارسال با `sendRichMessage` و فیلد مارکداون
- ویرایش پیام با `editMessageText` + `rich_message`
- ارسال پیش‌نویس با `sendRichMessageDraft` (۳۰ ثانیه، فقط چت خصوصی)
- روش جایگزین با `sendMessage` + `parse_mode=HTML`

### امکانات ویرایشگر
- صفحه دو قسمتی: ویرایشگر (۶۰٪) + پیش‌نمایش زنده (۴۰٪)
- پیش‌نمایش شبیه حباب پیام تلگرام
- تم تاریک (پیش‌فرض) + تم روشن
- نوار ابزار شناور با همه دکمه‌ها
- منوی درج بلوک
- کلیدهای میانبر (Ctrl+B, Ctrl+I, Ctrl+U, Ctrl+K, Ctrl+E)
- کشیدن و رها کردن فایل‌های رسانه
- شمارشگر کاراکتر (حداکثر ۳۲۷۶۸)
- پنل تنظیمات (توکن ربات، شناسه چت، روش ارسال)
- حالت فوکوس برای نوشتن بدون حواس‌پرتی
- دکمه پاک کردن همه
- اعلان‌های توست
- واکنش‌گرا برای موبایل

## معماری

```
DOM ویرایشگر (contenteditable)
    |
    v  (تبدیل‌کننده domToMarkdown)
  متن مارکداون
    |
    v  (API sendRichMessage)
تلگرام Bot API 10.1
    |
    v
  پیام غنی در چت
```

ویرایشگر از `contenteditable` برای ویرایش بصری استفاده می‌کنه. یه مبدل DOM به مارکداون محتوای ویرایشگر رو به مارکداون تلگرام تبدیل می‌کنه. پیام‌ها از طریق `sendRichMessage` با فیلد `markdown` ارسال می‌شن.

## نصب و اجرا

### پیش‌نیازها
- Node.js نسخه ۱۸ یا بالاتر
- یه ربات تلگرام (از @BotFather بگیرین)
- شناسه چت (کانال یا گروه)

### مراحل نصب

```bash
# ۱. مخزن رو کلون کنید
git clone https://github.com/Aporis3674/TelegramFreeRich.git
cd TelegramFreeRich

# ۲. وابستگی‌ها رو نصب کنید
npm install

# ۳. سرور توسعه رو اجرا کنید
npm run dev
```

### ساخت نسخه نهایی

```bash
# نسخه نهایی رو بسازید
npm run build

# خروجی در پوشه dist/
# می‌تونید با سرویس‌های رایگان مثل GitHub Pages یا Vercel منتشر کنید

# پیش‌نمایش نسخه ساخته شده
npm run preview
```

## نحوه استفاده

۱. اپ رو در مرورگر باز کنید
۲. روی تنظیمات (آیکون چرخ دنده) کلیک کنید
۳. توکن ربات رو از @BotFather وارد کنید
۴. شناسه چت رو وارد کنید (قالب @channel یا عدد)
۵. روش ارسال رو انتخاب کنید (sendRichMessage توصیه می‌شه)
۶. شروع به نوشتن در ویرایشگر کنید
۷. پیش‌نمایش زنده رو در سمت راست ببینید
۸. دکمه ارسال رو بزنید

## روش‌های ارسال

| روش | زمان استفاده |
|------|--------------|
| `sendRichMessage` (مارکداون) | توصیه می‌شه. همه قابلیت‌ها رو پشتیبانی می‌کنه |
| `sendRichMessage` (HTML) | قابلیت‌های کمتر. بدون پانویس و رسانه |
| `sendMessage` (parse_mode=HTML) | روش قدیمی. فقط فرمت‌های ساده |
